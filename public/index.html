üìñ API Overview

Welcome to the API documentation. This API provides endpoints for managing authentication, categories, products, warehouses, partners, units, and transactions.

Base URL

http://localhost:3000/api

Response Format

All responses follow a consistent JSON format:

{
  "success": true,
  "message": "Operation successful",
  "data": { ... }
}
GET /api/health

Health check endpoint to verify server status

Response (200): 
{
  "success": true,
  "message": "Server is running",
  "timestamp": "2024-01-01T00:00:00.000Z"
}
