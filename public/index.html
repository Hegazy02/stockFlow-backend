üìñ API Overview

Welcome to the API documentation. This API provides endpoints for managing authentication, categories, and products.

Base URL

http://localhost:3000/api

Response Format

All responses follow a consistent JSON format:

{
  "success": true,
  "message": "Operation successful",
  "data": { ... }
}
GET /api/health

Health check endpoint to verify server status

Response (200): 
{
  "success": true,
  "message": "Server is running",
  "timestamp": "2024-01-01T00:00:00.000Z"
}

üîê Authentication Endpoints

POST /api/auth/register

Register a new user account

Request Body:
Field Type Required Description
username string Required Username (3-30 characters)
email string Required Valid email address
password string Required Password (min 6 characters)
Example Request: 
{
  "username": "johndoe",
  "email": "john@example.com",
  "password": "securepass123"
}
Response 201 Created: 
{
  "success": true,
  "message": "User registered successfully",
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": "507f1f77bcf86cd799439011",
    "username": "johndoe",
    "email": "john@example.com",
    "createdAt": "2024-01-01T00:00:00.000Z"
  }
}

üìÅ Categories Endpoints

POST /api/categories

Create a new category with a unique name

Request Body:
Field Type Required Description
name string Required Category name (unique, max 100 chars)
description string Optional Category description (max 500 chars)
status string Optional "Active" or "Inactive" (default: "Active")
Example Request: 
{
  "name": "Electronics",
  "description": "Electronic devices and accessories",
  "status": "Active"
}
Response 201 Created: 
{
  "success": true,
  "message": "Category created successfully",
  "data": {
    "_id": "507f1f77bcf86cd799439011",
    "name": "Electronics",
    "description": "Electronic devices and accessories",
    "status": "Active",
    "createdAt": "2024-01-01T00:00:00.000Z",
    "updatedAt": "2024-01-01T00:00:00.000Z"
  }
}
Error Response 400 Bad Request: 
{
  "success": false,
  "message": "Category with this name already exists"
}
GET /api/categories

Get all categories with optional filtering, searching, and pagination

Query Parameters:
Parameter Type Required Description
status string Optional Filter by status (Active/Inactive)
search string Optional Search in name or description
page number Optional Page number (default: 1)
limit number Optional Items per page (default: 10)
Example Request: 
GET /api/categories?status=Active&search=tech&page=1&limit=10
Response 200 OK:

      "updatedAt": "2024-01-01T00:00:00.000Z"

    {
      "_id": "507f1f77bcf86cd799439012",
      "name": "Furniture",

    "total": 25,

    "pages": 3
  }

POST /api/categories/bulk-delete

Delete multiple categories in a single operation

Request Body:
Field Type Required Description
ids array Required Array of MongoDB ObjectIds (min 1 item)
Example Request: 
{
  "ids": [
    "507f1f77bcf86cd799439011",
    "507f1f77bcf86cd799439012",
    "507f1f77bcf86cd799439013"
  ]
}
Response 200 OK: 
{
  "success": true,
  "message": "3 categories deleted successfully",
  "data": {
    "deletedCount": 3
  }
}
Error Response 400 Bad Request: 
{
  "success": false,
  "message": "\"ids\" must contain at least 1 items"
}

üì¶ Products Endpoints

POST /api/products

Create a new product

Request Body:
Field Type Required Description
sku string Required Unique product SKU (auto-uppercase)
name string Required Product name (max 200 chars)
categoryId string Required MongoDB ObjectId of category
description string Optional Product description (max 1000 chars)
costPrice number Required Cost price (must be >= 0)
sellingPrice number Required Selling price (must be >= 0)
supplierId string Required MongoDB ObjectId of supplier
status string Optional "Active" or "Inactive" (default: "Active")
Example Request: 
{
  "sku": "PROD-001",
  "name": "Wireless Mouse",
  "categoryId": "507f1f77bcf86cd799439011",
  "description": "Ergonomic wireless mouse",
  "costPrice": 15.50,
  "sellingPrice": 29.99,
  "supplierId": "507f1f77bcf86cd799439012",
  "status": "Active"
}
Response 201 Created: 
{
  "success": true,
  "message": "Product created successfully",
  "data": {
    "_id": "507f1f77bcf86cd799439013",
    "sku": "PROD-001",
    "name": "Wireless Mouse",
    "categoryId": "507f1f77bcf86cd799439011",
    "description": "Ergonomic wireless mouse",
    "costPrice": 15.50,
    "sellingPrice": 29.99,
    "supplierId": "507f1f77bcf86cd799439012",
    "status": "Active",
    "createdAt": "2024-01-01T00:00:00.000Z",
    "updatedAt": "2024-01-01T00:00:00.000Z"
  }
}
